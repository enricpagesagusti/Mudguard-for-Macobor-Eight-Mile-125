# PETG Mudguard for Macobr Eight Mile 125
This repository contains the STL file for a printable PETG front mudguard or fender, designed for the Macbor Eight Mile 125.

This model has been stress tested and proven practically in off-road conditions as part of an Automotive's Master's Degree final project for the Universitat Politècnica de Catalunya.

The model is modular to reduce single print size, consisting off 3 distinct pieces that can be printed in most 3d printers, that interlock and are meant to be glued together for proper operation.

This model is open to the public and free to use, print, and modify for anyone who desires to use it.
