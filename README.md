# PETG Mudguard for Macobr Eight Mile 125
This repository contains the STL file for a printable PETG front mudguard or fender, designed for the Macbor Eight Mile 125.

This model has been stress tested and proven practically in off-road conditions as part of an Automotive's Master's Degree final project.

This model is free to use, print, and modify for anyone who desires to use it.ç
